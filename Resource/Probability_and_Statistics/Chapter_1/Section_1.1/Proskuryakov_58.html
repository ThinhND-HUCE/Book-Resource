Cách 1: Tính trực tiếp theo quy tắc tam giác: \[ \begin{vmatrix} 0 & a & 0 \\ b & c & d \\ 0 & e & 0 \end{vmatrix} = 0\left( c \right)\left( 0 \right) + a\left( d \right)\left( 0 \right) + b\left( e \right)\left( 0 \right) - 0\left( c \right)\left( 0 \right) - b\left( a \right)\left( 0 \right) - e\left( d \right)\left( 0 \right) = 0 \] Cách 2: Khai triển theo hàng 1 (có nhiều phần tử 0): \[ \begin{vmatrix} 0 & a & 0 \\ b & c & d \\ 0 & e & 0 \end{vmatrix} = a\times \left( -1 \right)^{1 + 2} \times \begin{vmatrix} b & d \\ 0 & 0 \end{vmatrix} = -a\left( b\times 0 - 0\times d \right) = 0 \]
