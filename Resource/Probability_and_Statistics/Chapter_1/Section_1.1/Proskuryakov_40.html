Bài 40:

Nếu \(a = 0\) thì để \(2bx + c\) là đa thức chính phương, điều kiện cần và đủ là \(b = 0\), tức là \(\begin{vmatrix} a & b \\ b & c \end{vmatrix} = ac - b^2 = 0\). Như vậy ta chỉ cần chứng minh cho trường hợp \(a\ne 0\).
Điều kiện cần: Giả sử \(ax^2 + 2bx + c = (px + q)^2 = p^2 x^2 + 2pqx + q^2\). Khi đó \(a = p^2\), \(2b = 2pq\Rightarrow b = pq\), và \(c = q^2\). Suy ra \(\begin{vmatrix} a & b \\ b & c \end{vmatrix} = ac - b^2 = p^2 q^2 - \left(pq\right)^2 = 0\)
Điều kiện đủ: Giả sử \(\begin{vmatrix} a & b \\ b & c \end{vmatrix} = ac - b^2 = 0\). Ta có \(\displaystyle ax^2 + 2bx + c = a\left( {{x^2} + 2\frac{b}{a}x} \right) + c = a\left[ {{x^2} + 2\frac{b}{a}x + {{\left( {\frac{b}{a}} \right)}^2}} \right] - a{\left( {\frac{b}{a}} \right)^2} + c = a{\left( {x + \frac{b}{a}} \right)^2} + \frac{{ac - {b^2}}}{a} = a{\left( {x + \frac{b}{a}} \right)^2}\)
Gọi \(a_0\) là một căn bậc hai của \(a\). Ta có \(a_0^2 = a\), và \[ ax^2 + 2bx + c = {a_0^2}{\left( {x + \frac{b}{{{a_0^2}}}} \right)^2} = {\left( {a_0 x + \frac{b}{a_0}} \right)^2} \] là một đa thức chính phương. Ta có điều phải chứng minh.
