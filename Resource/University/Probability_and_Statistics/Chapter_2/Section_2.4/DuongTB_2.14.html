Bài 2.14:

Hai đại lượng ngẫu nhiên \(X\) và \(Y\) có hàm mật độ xác suất tương ứng là \[ {f_1}\left( x \right) = \begin{cases} 2\ln 3 \cdot {3^{ - 2x}} & \text{khi } x\ge 0 \\ 0 & \text{khi } x < 0 \end{cases}\quad \text{và} \quad f_2\left( y \right) = \begin{cases} 5e^{-5y} & \text{khi } y\ge 0 \\ 0 & \text{khi } y < 0 . \end{cases} \] Tìm \(E\left( X+Y \right)\).

Đáp số:
\(EX = \dfrac{1}{{2\ln 3}}\), \(EY = \dfrac{1}{5}\), \(E\left( X+Y \right) = EX + EY\).
