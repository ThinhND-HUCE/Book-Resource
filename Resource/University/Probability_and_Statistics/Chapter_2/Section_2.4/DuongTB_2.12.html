Bài 2.12:

Hàm phân bố của đại lượng ngẫu nhiên \(X\) có dạng \[ F(x) = \begin{cases} 1 & \text{nếu } x\ge a \\ A + B\arcsin \dfrac{x}{a} & \text{nếu } -a < x < a \\ 0 & \text{nếu } x\le -a. \end{cases} \]
  1. Xác định \(A, B\) sao cho \(F\) liên tục trên \(\Bbb R\).
  2. Tính \(P\left( { - \dfrac{a}{2} < X < \dfrac{a}{2}} \right)\).
  3. Tìm mật độ phân bố xác suất của \(X\).
  4. Tính \(EX, DX, \sigma(X)\).
Đáp số:
  1. \(A = \dfrac{1}{2}, B = \dfrac{1}{\pi}\)
  2. \(\dfrac{1}{3}\)
  3. \(f\left( x \right) = F'\left( x \right) = \begin{cases} \dfrac{1}{{\pi \sqrt {{a^2} - {x^2}} }} & \text{nếu } -a < x < a \\ 0 & \text{nếu } \left| x \right| \ge a \end{cases}\)
  4. 0, \(\dfrac{{{a^2}}}{2}\), \(\dfrac{a}{{\sqrt 2 }}\)
