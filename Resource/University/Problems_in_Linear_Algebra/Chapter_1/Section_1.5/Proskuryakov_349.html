Bài 349:

Trước hết ta chứng minh \(\cos n\varphi \), với \(n\) nguyên dương bất kỳ, là đa thức bậc \(n\) theo \(\cos\varphi\), có hệ số cao nhất (hệ số của \(\cos^n\varphi\)) bằng \(2^{n-1}\). Thật vậy, theo công thức Moivre, \({\left( {\cos \varphi + i\sin \varphi } \right)^n} = \cos n\varphi + i\sin n\varphi \), khai triển lũy thừa theo công thức nhị thức Newton, ta có \[ \cos n\varphi + i\sin n\varphi = \sum\limits_{k = 0}^n {C_n^k{{\left( {\cos \varphi } \right)}^{n - k}}{{\left( {i\sin \varphi } \right)}^k}} , \] và tương tự \[ \cos n\varphi - i\sin n\varphi = \sum\limits_{k = 0}^n {C_n^k{{\left( {\cos \varphi } \right)}^{n - k}}{{\left( { - i\sin \varphi } \right)}^k}} . \] Cộng hai vế của các đẳng thức trên, suy ra \[ 2\cos n\varphi = \sum\limits_{k = 0}^n {C_n^k\left( {{{\cos }^{n - k}}\varphi } \right)\left[ {{{\left( {i\sin \varphi } \right)}^k} + {{\left( { - i\sin \varphi } \right)}^k}} \right]} . \] Trong tổng trên, với \(k\) lẻ thì số hạng bằng 0, nên ta chỉ cần xét các số hạng với chỉ số chẵn (thay chỉ số chạy \(k\) bởi \(2k\)): \begin{align} \cos n\varphi & = \frac{1}{2}\sum\limits_{k = 0}^{\left\lfloor {\frac{n}{2}} \right\rfloor } {C_n^{2k}\left( {{{\cos }^{n - 2k}}\varphi } \right)\left[ {{{\left( {i\sin \varphi } \right)}^{2k}} + {{\left( { - i\sin \varphi } \right)}^{2k}}} \right]} \\ & = \sum\limits_{k = 0}^{\left\lfloor {\frac{n}{2}} \right\rfloor } {{{\left( { - 1} \right)}^k}C_n^{2k}\left( {{{\cos }^{n - 2k}}\varphi } \right){{\left( {{{\sin }^2}\varphi } \right)}^k}} = \sum\limits_{k = 0}^{\left\lfloor {\frac{n}{2}} \right\rfloor } {{{\left( { - 1} \right)}^k}C_n^{2k}\left( {{{\cos }^{n - 2k}}\varphi } \right){{\left( {1 - {{\cos }^2}\varphi } \right)}^k}} \\ & = \sum\limits_{k = 0}^{\left\lfloor {\frac{n}{2}} \right\rfloor } {C_n^{2k}\left( {{{\cos }^{n - 2k}}\varphi } \right){{\left( {{{\cos }^2}\varphi - 1} \right)}^k}} \end{align} Hệ số của \(\cos n\varphi\) trong biểu thức trên là \[ \sum\limits_{k = 0}^{\left\lfloor {\frac{n}{2}} \right\rfloor } {C_n^{2k}} = C_n^0 + C_n^2 + C_n^4 + \cdots \] Bây giờ ta chứng minh tổng này bằng \(2^{n-1}\). Xét khai triển nhị thức \[ {\left( {x + y} \right)^n} = \sum\limits_{k = 0}^n {C_n^k{x^{n - k}}{y^k}} = C_n^0{x^n} + C_n^1{x^{n - 1}}y + C_n^2{x^{n - 2}}{y^2} + \cdots + C_n^n{y^n} . \] Trong khai triển trên, cho \(x = y = 1\), thì \[ C_n^0 + C_n^1 + C_n^2 + \cdots + C_n^n = 2^n, \] và cho \(x = 1, y = -1\), thì \[ C_n^0 - C_n^1 + C_n^2 + \cdots + \left( -1 \right)^n C_n^n = 0, \] Cộng hai vế của hai đẳng thức trên, ta sẽ được \(C_n^0 + C_n^2 + C_n^4 + \cdots = 2^{n-1}\), là hệ số của \(\cos n \varphi\) khi biểu diễn đa thức theo \(\cos\varphi\).
Đặt \(f_n\left( \cos \varphi \right)\) là đa thức biểu diễn theo \(\cos\varphi\) của \(\cos n\varphi\), thì \[ f_n\left( x \right) = 2^{n-1} x^n + \left( * \right) x^{n-1} + \cdots + \left( * \right), \] trong đó các hệ số \(*\) trong dấu ngoặc, như lập luận sau này, không ảnh hưởng đến kết quả của định thức \(D\) đã cho. Ta có \[ D = \begin{vmatrix} 1 & {f_1}\left( {\cos {\varphi _1}} \right) & {f_2}\left( {\cos {\varphi _1}} \right) & \ldots & {f_{n-1}}\left( {\cos {\varphi _1}} \right) \\ 1 & {f_1}\left( {\cos {\varphi _2}} \right) & {f_2}\left( {\cos {\varphi _2}} \right) & \ldots & {f_{n-1}}\left( {\cos {\varphi _2}} \right) \\ \ldots & \ldots & \ldots & \ldots & \ldots \\ 1 & {f_1}\left( {\cos {\varphi _n}} \right) & {f_2}\left( {\cos {\varphi _n}} \right) & \ldots & {f_{n-1}}\left( {\cos {\varphi _n}} \right) \end{vmatrix} \] Đây là định thức được xét trong Bài tập 335, ta có \[ D = {2^{0 + 1 + \cdots + \left( {n - 2} \right)}} \prod\limits_{i > j} {\left( {{\cos\varphi_i} - {\cos\varphi_j}} \right)} = {2^{\frac{{\left( {n - 1} \right) \left( {n - 2} \right)}}{2}}} \prod\limits_{i > j} {\left( {{\cos\varphi_i} - {\cos\varphi_j}} \right)} . \]
