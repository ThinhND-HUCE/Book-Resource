Bài 220:

Theo Bài tập 218, \(\overline{D} = \left( -i \right)^n D\). Nếu \(n\) lẻ, tức là \(n = 2m+1\), \(m\in\Bbb N\), thì \(\overline{D} = \left( -i \right)^{2m+1} D = \left[ \left( -i \right)^2 \right]^m iD = \left( -1 \right)^m iD\). Giả sử \(D = a + ib\), với \(a, b\in \Bbb R\), ta có \(\overline D = a - ib\), suy ra \begin{align} & a - ib = \left( -1 \right)^m i\left( a + ib \right) = \left( -1 \right)^m \left( ia + i^2 b\right) = \left( -1 \right)^m \left( -b + ia \right) \\ \Rightarrow\; & a = \left( -1 \right)^m \left( -b \right)\quad \text{và}\quad -b = \left( -1 \right)^m a \\ \Rightarrow\; & b = - \left( -1 \right)^m a = \pm a \quad \text{(tùy theo \(m\) lẻ hay chẵn)} \end{align} Vậy định thức có dạng \(a\left( 1\pm i \right)\).
