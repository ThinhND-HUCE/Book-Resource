  • Nếu \(\sin\left( \alpha + \beta \right) = 0\), hay \(\alpha + \beta = k\pi\), \(k \in \Bbb Z\), thì \(0x = \sin\alpha + \sin\beta\)
    • Nếu \(\sin\alpha + \sin\beta = 0 \Leftrightarrow \sin\alpha = - \sin\beta = \sin\left( -\beta \right)\Leftrightarrow \alpha = -\beta + \ell 2\pi\) hoặc \(\alpha = \pi - \left( -\beta \right) + \ell 2\pi\), \(\ell \in \Bbb Z\), thì phương trình có vô số nghiệm
      Kết hợp \(\alpha = -\beta + \ell 2\pi\) với \(\alpha + \beta = k\pi\) suy ra \(k = 2\ell\)
      Kết hợp \(\alpha = \beta + \left( 2\ell + 1 \right) \pi\) với \(\alpha + \beta = k\pi\) suy ra \(\beta = \dfrac{\left( k - 2\ell - 1\right)\pi}{2}\) và \(\alpha = \dfrac{\left( k + 2\ell + 1\right)\pi}{2}\)
    • Nếu \(\alpha \ne -\beta + \ell 2\pi\) và \(\alpha \ne \beta + \left( 2\ell + 1 \right)\pi \), thì phương trình vô nghiệm.
      Vì \(\alpha + \beta = k\pi\) và \(\alpha \ne -\beta + \ell 2\pi\) nên \(\alpha = -\beta + \left( 2k' + 1 \right)\pi\), \(k'\in\Bbb Z\). Mặt khác \(\alpha \ne \beta + \left( 2\ell + 1 \right)\pi \) nên \( - \beta + \left( {2k' + 1} \right)\pi \ne \beta + \left( {2\ell + 1} \right)\pi \Rightarrow 2\beta \ne 2\left( {k' - \ell} \right)\pi \Rightarrow \beta \ne \left( {k' - \ell} \right)\pi = m\pi \), \(m\in\Bbb Z\)
  • Nếu \(\alpha + \beta \ne k\pi\) thì phương trình có nghiệm duy nhất
